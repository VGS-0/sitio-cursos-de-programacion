Sobre el curso

Aprende diseño web desde cero con actividades prácticas y divertidas. En este curso verás:

  • HTML y CSS desde lo básico
  • Diseño visual moderno
  • Edición de imágenes
  • Diseño adaptable para celulares
  • Proyectos reales paso a paso

¿Qué lograrás al final del curso?

Al finalizar este curso, serás capaz de crear sitios web modernos, adaptativos y visualmente atractivos, desarrollando habilidades prácticas en:

  • Maquetación con HTML5 y CSS3
  • Diseño responsivo
  • Uso de Bootstrap
  • Optimización de imágenes
  • Accesibilidad y usabilidad
  • Publicación de sitios web
